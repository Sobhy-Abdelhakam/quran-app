# quran-app

**Quran Application contain:**

* Al-Madinah Al-Munawwarah edition
* Al-Shamrali Edition (Egyptian Edition)
* Tafaseer of Quran
* A large number of reciters

**Libraries & technologies used**

* RecyclerView
* ViewPager2
* MVVM as architecture patterns
* Room database
* Retrofit
* Notifications Manager
* Service
* Download Manager
* MediaPlayer
